# 2026-04-26 日历内后台任务清单入口 Worklog

## 用户反馈

后台任务已经能安静挂在后台，但用户希望点开「打开日历」后，有一个可以查看所有任务的选项。打开后需要看到：

- 有哪些任务清单。
- 每个任务的基本名字。
- 每个任务的详细日期信息。
- 里程碑、子任务、候选写入日历的时间块。

## 本次完成

- 重构 `CalendarPanel` 为双视图：
  - 「日历」：原日程视图。
  - 「查看所有任务」：后台任务清单视图。
- 打开日历时同时加载：
  - 日历事件。
  - 待确认安排。
  - 后台任务 `background_tasks`。
- 任务清单左侧展示：
  - 任务名称。
  - 日期范围 / 首个候选日期。
  - 任务类型。
  - 状态。
- 任务详情右侧展示：
  - 任务标题和目标。
  - 日期范围。
  - 来源 Agent。
  - 任务类型。
  - 里程碑及日期。
  - 子任务及日期。
  - 建议写入日历的时间块。
- 保留原日历新增、修改、删除、待确认安排确认/修改/取消能力。

## 产品体验方式

1. 在 Jarvis 首页点击「打开日历」。
2. 顶部选择「查看所有任务」。
3. 左侧看到所有后台任务。
4. 点击某个任务。
5. 右侧查看该任务的日期范围、里程碑、子任务和候选日历时间块。
6. 点击「日历」可回到原来的日程视图。
