# 路演用户画像种子：雅思备考 + 生活节律 + 暑假旅行

创建时间：2026-04-26（Asia/Shanghai）

## 1. 用途

这个文件用于路演和回归测试前加载一个固定、鲜明、可追溯的人物画像，避免平时测试产生的脏数据影响演示效果。

目标效果：

- 每次路演从同一个用户状态开始。
- Agent 的关心、日程安排、任务拆解和记忆提取都围绕同一个人物画像。
- 每个 demo step 都可以追溯到画像、输入、工具调用和写入结果。

## 7. 期望 Agent 行为

- Maxwell：负责长期任务拆解、日程安排、冲突检查和后台任务清单。
- 心理管理员：关注焦虑、压力和自我要求过高，但不做医疗诊断。
- 生活节律管理员：关注睡眠、学习块、休息和节律稳定性。
- 其它 Agent：可以识别任务/日程意图，但长期任务应转交 Maxwell。
