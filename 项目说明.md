# Jarvis-Life / ShadowLink 项目说明

> 初始分析时间：2026-04-24 18:40（Asia/Shanghai）

## 1. 产品定位

Jarvis-Life 当前仓库包含两个形态的产品实现：

1. **PyQt 桌面端一体化应用**：根目录 `launcher.py` / `main.py` 是现有桌面客户端入口，集成 LLM 对话、RAG 检索、任务/模式管理、本地资源打开、历史记录等能力。
2. **ShadowLink 分层架构版本**：由 `shadowlink-web`、`shadowlink-electron`、`shadowlink-server`、`shadowlink-ai`、`deploy`、`proto` 等模块组成，目标是把前端、桌面壳、Java 后端、Python AI 服务、协议与部署拆分成可独立演进的系统。

整体产品方向是个人 AI 工作台 / 本地生活与知识助理：通过模型对话、知识库检索、Agent 工具调用、多 Agent 协作、文件解析、本地资源管理和工作模式管理，帮助用户完成问答、资料整理、任务推进和上下文记忆。

## 2. 核心功能概览

### 2.1 AI 对话与模型接入

- 支持 OpenAI 兼容接口、DeepSeek、Ollama、本地模型等提供商接入。
- 支持流式输出、模型配置读取、模型切换和调用异常处理。
- 根目录桌面版主要由 `llm/llm_client.py` 与兼容转发文件 `llm_client.py` 提供能力。
- 分层 AI 服务版由 `shadowlink-ai/app/llm/` 下的统一客户端、Provider、缓存、限流、Token 统计等模块提供能力。

### 2.2 RAG 知识库检索

- 支持本地文档采集、切分、向量化、索引构建和检索问答。
- 根目录桌面版主要由 `rag/rag_engine.py`、`rag/web_crawler.py` 和兼容转发文件 `rag_engine.py` 提供能力。
- 分层 AI 服务版由 `shadowlink-ai/app/rag/` 提供更完整的 RAG 流水线，包括 chunking、embedding、FAISS/Milvus 索引、BM25/向量/混合检索、多查询、自反式检索和重排序。

### 2.3 Agent 与工具调用

- 根目录桌面版通过 `core/skill_interface.py`、`tools/my_skills.py` 装配本地技能。
- AI 服务版提供 ReAct、Plan-and-Execute、Brainstorm、多 Agent、记忆模块、工具注册和 MCP 适配。
- 内置工具覆盖系统工具、文件操作、代码执行、本地搜索、知识库搜索、网页搜索、Jarvis 专用工具等。

### 2.4 桌面工作台与本地资源

- PyQt 桌面端提供主窗口、配置弹窗、模型设置、本地资源管理、任务卡片、RAG 索引和聊天线程。
- `main.py` 中的 `FocusFlow` 是桌面主窗口核心类。
- 支持打开文件/目录/链接、检测本地启动方式、保存模式配置和运行外部命令。
- `assets/` 提供图标与图片资源。

### 2.5 Web / Electron 客户端

- `shadowlink-web` 是 React + Vite + TypeScript 前端，承担浏览器端交互、聊天页面、组件化 UI、主题和状态管理。
- `shadowlink-electron` 是 Electron 桌面壳，目标是提供托盘、全局热键、剪贴板和原生桌面集成。

### 2.6 Java 后端与业务服务

- `shadowlink-server` 是 Maven 多模块 Spring Boot 后端。
- 主要能力包括认证、会话、工作模式、网关代理、WebSocket、AI 服务桥接、通用基础设施和启动器。
- 入口在 `shadowlink-server/shadowlink-starter/src/main/java/com/shadowlink/ShadowLinkApplication.java`。

### 2.7 部署与协议

- `deploy/` 提供 Docker Compose、Nginx、MySQL、Redis、Prometheus 和初始化 SQL。
- `proto/` 定义 Agent、MCP、Memory、RAG 等服务的 gRPC/Protobuf 协议。

## 4. PyQt 桌面版代码说明

### 4.1 启动链路

- `launcher.py`：导入 `main.run_app` 并在脚本入口调用，是源代码运行和打包时的统一入口。
- `main.py:run_app()`：创建 `QApplication`、加载图标/字体、初始化主窗口并进入 Qt 事件循环。
- `main.py:FocusFlow`：主窗口类，承载主要页面、交互状态、聊天/RAG/任务等核心 UI 逻辑。

## 7. `shadowlink-web` 前端说明

- React 18 + TypeScript + Vite。
